# **App Name**: Ese's Portfolio

## Core Features:

- About Me Section: An introductory section highlighting Ese's background as a Full-Stack web developer, including her recent training at the Digital Career Institute (DCI) and her soft skills.
- Skills Showcase: A dedicated section showcasing Ese's technical skills (HTML, CSS, JavaScript, React, Node.js, etc.) and tools/deployment experience (Git, GitHub, Vercel, etc.).
- Projects Portfolio: A gallery showcasing personal and DCI projects, with descriptions and links to the GitHub repositories or live demos. A tool decides on what project details (title, image, description, github link) would be most compelling for this potential employer.
- Experience Timeline: A timeline-based display of Ese's educational background and professional experience, emphasizing roles and responsibilities.
- Contact Form: A simple form for potential employers or clients to send messages directly to Ese's email.
- Blog Section: Dedicated Blog section where she uploads articles and project briefs.

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA) to convey femininity and professionalism.
- Background color: Light Peach (#FAF0E6) for a warm and inviting feel.
- Accent color: Dusky Rose (#BC8F8F) to highlight key sections and create visual interest.
- Body font: 'Literata', serif, to be professional but with a personal touch. 
- Headline font: 'Belleza', sans-serif, to give personality and fashion sense.
- Use minimalist icons for navigation and skill representation.
- Clean, single-page layout with clear section dividers for easy navigation.
